# TechNova-Network

Proyecto de red empresarial implementado en Cisco Packet Tracer.

## Descripción
Diseño de una red LAN corporativa segmentada por departamentos que integra:
- WiFi (Marketing)
- Departamentos cableados (Administración y RR.HH)
- Smart Room con dispositivos IoT
- Sala de servidores con DHCP, DNS, WEB e IoT

## Servicios configurados
| Servicio | IP |
|--------|--------|
| DHCP | 192.168.1.2 |
| IoT | 192.168.1.3 |
| WEB | 192.168.1.4 |
| DNS | 192.168.1.5 |

## Pruebas
- DHCP operativo en PCs cableados
- Acceso web por IP y por DNS
- Comprobación de conectividad entre dispositivos de diferentes áreas
